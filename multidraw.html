Multidraw   Canvas

Welcome to my Multidraw Canvas. This is my very first project created utilising both PHP+AJAX and HTML5 canvas.

This small application picks up end points of line segments being drawn and stores them in a so-called swap file on remote server. Regular-timed update fetches the data from the server and draws given lines. The swap file is only one on the server, therefore there is only one instance of the Multidraw Canvas running. That means everybody around the globe is drawing in the very same canvas!

controls:
- click to choose points in the canvas for lines to be drawn
- move cursor out of the canvas to break the line
- double click to wipe the canvas (for everyone!)
